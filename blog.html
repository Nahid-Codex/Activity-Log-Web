Question-1: What are the different ways to select an element in the DOM?

Many way to select an element in the DOM , you can use methods like getElementById , getElementsByClassName , getElementsByTagName & querySelector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML output the HTML code & contant including tags, innerText Gives only the text content of an element , and textContent gives the text content, including all text (visible or hidden) without HTML tags. innerHTML works with HTML, while innerText and textContent deal with plain text, but textContent is faster than innerText.

Question-3: What is event delegation in the DOM?

Event delegation is a design pattern in JavaScript used to efficiently manage and handle events on multiple child elements by attaching a single event listener to each child element.

Question-4: What is event bubbling in the DOM?

Event Bubbling is the behavior where when an event occurs on an element is clicked, the event happens on the button first. Then, it moves up to the parent and keeps going up to the top. This is called event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

Create new element using document.createElement() and then add it to the DOM with appendChild() or insertBefore(). To remove an element useing removeChild() or parentNode.removeChild() on its parent element.
